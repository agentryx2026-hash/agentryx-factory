## 1. AI-Driven Manual Generation Strategy

### The Problem with Manual Documentation
Humans write docs → UI changes → docs are outdated → nobody updates them → docs become useless. **AI eliminates this cycle** — the agent generates docs FROM the live application, every time.

### How It Works — The Pipeline

```
Step 1: CONTENT GENERATION (AI Agent)
  │  Agent reads: A3 (Modules), A5 (PRD), B7 (Admin Ops), route files, UI components
  │  Agent generates: Feature descriptions, step-by-step instructions per role
  │  Output: manual_content.json (structured content per role)
  │
Step 2: SCREENSHOT CAPTURE (AI Agent + Playwright)
  │  Agent opens each page in headless Chromium via Playwright
  │  Agent navigates to each feature, captures full-page screenshots
  │  Agent adds annotation overlays (numbered callouts, red circles, arrows)
  │  Output: screenshots/ folder with annotated PNGs
  │
Step 3: PDF GENERATION (AI Agent + Puppeteer)
  │  Agent builds HTML manual template (branded, paginated, table of contents)
  │  Agent injects content + screenshots into HTML template
  │  Agent renders HTML → PDF via Puppeteer (pixel-perfect print layout)
  │  Output: C2_1_Candidate_Manual.pdf, C2_3_Admin_Manual.pdf, etc.
  │
Step 4: QUICK REFERENCE CARD (AI Agent + Puppeteer)
  │  Agent generates single-page HTML card per role
  │  Agent renders HTML → PDF (single A4 page, printable)
  │  Output: C2_5_Quick_Reference_{Role}.pdf
  │
Step 5: IN-APP HELP PAGES (AI Agent)
  │  Agent generates React components with help content
  │  Help content is embedded in the application itself
  │  Output: client/src/components/help/ folder
```

## 3. Screenshot Capture Process

### Automated Screenshot Script
```typescript
// scripts/capture-manual-screenshots.ts
// AI Agent generates and runs this script

import { chromium } from 'playwright';

async function captureScreenshots() {
  const browser = await chromium.launch();
  const context = await browser.newContext({
    viewport: { width: 1440, height: 900 },
    // Use realistic demo data session
    storageState: 'demo-session.json', // Pre-authenticated session
  });
  const page = await context.newPage();

  // === CAPTURE: Dashboard ===
  await page.goto('https://staging.domain.dev/dashboard');
  await page.waitForLoadState('networkidle');
  
  // Add annotation overlay before screenshot
  await annotate(page, [
    { selector: '.sidebar-nav', label: '①', text: 'Navigation Menu' },
    { selector: '.stats-cards', label: '②', text: 'Key Statistics' },
    { selector: '.recent-activity', label: '③', text: 'Recent Activity' },
  ]);
  
  await page.screenshot({ 
    path: 'screenshots/dashboard_annotated.png',
    fullPage: false, // Viewport only (not scroll)
  });

  // === CAPTURE: Registration Form (with field labels) ===
  await page.goto('https://staging.domain.dev/register');
  await annotate(page, [
    { selector: '#name', label: '①', text: 'Full Name (required)' },
    { selector: '#email', label: '②', text: 'Email Address (required)' },
    { selector: '#phone', label: '③', text: 'Phone Number' },
    { selector: '#password', label: '④', text: 'Password (min 8 chars)' },
    { selector: 'button[type=submit]', label: '⑤', text: 'Click to submit' },
  ]);
  await page.screenshot({ path: 'screenshots/register_annotated.png' });

  // ... repeat for every page/feature

  await browser.close();
}

// Annotation helper — injects visual callouts on the page
async function annotate(page, annotations) {
  await page.evaluate((items) => {
    items.forEach(({ selector, label, text }) => {
      const el = document.querySelector(selector);
      if (!el) return;
      const rect = el.getBoundingClientRect();
      
      // Add red circle with number
      const circle = document.createElement('div');
      circle.style.cssText = `
        position: fixed; z-index: 99999;
        left: ${rect.left - 15}px; top: ${rect.top - 15}px;
        width: 30px; height: 30px; border-radius: 50%;
        background: #ef4444; color: white;
        display: flex; align-items: center; justify-content: center;
        font-weight: bold; font-size: 16px; font-family: Arial;
        box-shadow: 0 2px 8px rgba(0,0,0,0.3);
      `;
      circle.textContent = label;
      document.body.appendChild(circle);
      
      // Add red border around element
      el.style.outline = '2px solid #ef4444';
      el.style.outlineOffset = '2px';
    });
  }, annotations);
  
  // Let annotations render
  await page.waitForTimeout(500);
}
```

### PDF Generation Command
```typescript
// scripts/generate-manual-pdf.ts

import puppeteer from 'puppeteer';

async function generateManualPDF(role: string) {
  const browser = await puppeteer.launch();
  const page = await browser.newPage();
  
  // Load the populated HTML template
  await page.goto(`file:///path/to/manual_${role}.html`, {
    waitUntil: 'networkidle0'
  });
  
  // Generate PDF with print-quality settings
  await page.pdf({
    path: `C2_${role}_Manual.pdf`,
    format: 'A4',
    printBackground: true,
    margin: { top: '2cm', bottom: '2cm', left: '2cm', right: '2cm' },
    displayHeaderFooter: true,
    headerTemplate: '<div style="font-size:10px;text-align:center;width:100%;color:#666;">{Project Name} — User Manual</div>',
    footerTemplate: '<div style="font-size:10px;text-align:center;width:100%;">Page <span class="pageNumber"></span> of <span class="totalPages"></span></div>',
  });
  
  await browser.close();
  console.log(`✅ Generated: C2_${role}_Manual.pdf`);
}
```

## 8. Re-Generation Policy

| Trigger | Action | What's Regenerated |
|---------|--------|-------------------|
| UI redesign | Full re-generation | All screenshots + all PDFs |
| Single feature change | Partial re-generation | Affected screenshots + affected PDF sections |
| New feature added | Incremental | New screenshots + new section appended to PDF |
| New language needed | Content translation | All PDFs regenerated with translated text |
| Content correction | Targeted update | Specific section re-rendered |

### Why This Matters
> A human-written manual becomes outdated the moment the UI changes. An AI-generated manual can be **regenerated in under 5 minutes** — capture new screenshots, rebuild the PDF, done. The manual is always synchronized with the actual application.
