## 8. Troubleshooting

| Problem | Cause | Solution |
|---------|-------|---------|
| `ECONNREFUSED` on DB | Database not running | Start PostgreSQL or Docker |
| `JWT_SECRET undefined` | Missing .env | Copy `.env.example` to `.env` |
| Port already in use | Another process on port | Change PORT in .env or kill process |
| Module not found | Missing dependencies | Run `npm install` |
